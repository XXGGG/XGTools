# Changelog

本文档记录项目的所有重要变更。

格式参考 [Keep a Changelog](https://keepachangelog.com/)，版本号遵循 [语义化版本](https://semver.org/)。

## [Unreleased]

### Fixed
- 截图翻译后，选区外鼠标指针从 `text` 改为 `default`（默认箭头）
- 截图进行中重复按快捷键导致无限套娃，现改为取消当前截图
- 截图窗口吸附无法识别自身应用窗口（xcap 库过滤同进程窗口），改用 Win32 API 补充枚举
- 窗口吸附优先级改为按面积排序，小窗口优先于全屏窗口

### Added
- 截图工具栏新增翻译按钮，无需开启截图翻译快捷键也能直接翻译
- 截图翻译模式下，右键可关闭翻译覆盖层
- 截图标注画完后自动选中，可直接通过样式面板修改颜色、线宽等属性
- 点击已有标注可重新选中并修改样式
- 选中标注后可拖拽移动、拖拽控制点缩放大小
- 绘制/缩放时按住 Shift 可约束为正方形、正圆、45° 线条

## [v0.1.0] - 2026-02-23

### Added
- 暗黑模式 & 暗黑模式持久化
- 开机自启
- 按键显示模块（智能布局、躲避鼠标、自动清除、定位持久化）
- Windows 启动台
- 截图功能（选区截图、标注、投影背景、窗口圆角）
- 截图翻译功能
- 格式转换（图片/音视频）
- Todo 与计时器
- FFmpeg 缺失自动检测与一键下载
