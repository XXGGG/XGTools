## Git 提交规范 (Commit Convention)

为了保持提交历史清晰，请遵循以下格式：
`Emoji <type>: <Chinese description>`

**类型对照表**:
| Emoji | Type | Description |
| :--- | :--- | :--- |
| ✨ | feat | 新功能 |
| 🐛 | fix | 修复 Bug |
| ♻️ | refactor | 代码重构 (不影响功能) |
| 🎨 | style | 样式/格式调整 |
| 🧪 | test | 测试相关 |
| 📄 | docs | 文档更新 |
| 🔧 | chore | 构建/工具/配置修改 |
| 🚀 | perf | 性能优化 |
| 🔙 | revert | 回退提交 |
| 🛠 | build | 构建系统修改 |
| 📦 | deps | 依赖更新 |
| 🚨 | ci | CI 配置 |
