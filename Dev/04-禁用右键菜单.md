## 禁用全局右键菜单

为了让应用看起来更像原生应用，我们需要禁用网页默认的右键菜单（防止用户不小心点出“检查元素”等浏览器菜单）。

打开 `src/main.ts`，在最后添加以下代码：

```typescript
import { createApp } from "vue";
import App from "./App.vue";
import './style.css'

createApp(App).mount("#app");

// 禁用所有窗口的右键菜单
document.addEventListener('contextmenu', (event) => {
  event.preventDefault();
});
```

这样，无论是主窗口还是新建的“按键显示”窗口，右键点击都不会再弹出浏览器菜单了。
