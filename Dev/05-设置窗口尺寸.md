## 设置窗口最小尺寸

为了防止用户把窗口拖得太小导致布局错乱，我们可以在 `src-tauri/tauri.conf.json` 中设置窗口的最小宽度和高度。

打开 `src-tauri/tauri.conf.json`，找到 `app` -> `windows` 配置：

```json
"windows": [
  {
    "label": "main",
    "title": "XGTools",
    "width": 800,
    "height": 600,
    "minWidth": 800,   // <--- 添加这行
    "minHeight": 600,  // <--- 添加这行
    "decorations": false,
    "shadow": true
  }
],
```

这样用户就无法把窗口缩小到 800x600 以下了，保证了界面的完整性。
