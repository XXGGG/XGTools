## 1. 组件库 (Shadcn-vue)

**通用安装公式**:
```bash
pnpm dlx shadcn-vue@latest add <component-name>

##常用组件安装示例##:
# 按钮
pnpm dlx shadcn-vue@latest add button
# 输入框
pnpm dlx shadcn-vue@latest add input
# 卡片
pnpm dlx shadcn-vue@latest add card
# 弹窗 (Dialog)
pnpm dlx shadcn-vue@latest add dialog
```

## 2. 品牌标识 (Branding)
- **Logo 存放位置**: `src-tauri/icons`
- **图标规格与生成**:
  1. 准备一张 `512x512` 像素的 PNG 图片 (例如 `app-icon.png`)。
  2. 运行命令自动生成所有格式 (ico, icns, png):
      这个命令会自动生成并覆盖 `src-tauri/icons` 文件夹下的所有文件。
      ```bash
      pnpm tauri icon app-icon.png
      ```
  3. **注意**: Windows 必须要有 `icon.ico`，macOS 必须要有 `icon.icns`。

**关于托盘图标：**
 (`lib.rs`) 中：
```rust
.icon(app.default_window_icon().unwrap().clone())
```
这行代码使用的是应用的主图标（即 `icon.ico` 或 `icon.png`）。如果你想为托盘单独指定一个图标（例如一个更简单的单色图标），可以把那个图标放在 `icons` 文件夹下（例如 `tray-icon.png`），然后在代码里加载它。但在大多数情况下，直接使用主图标就足够了。
