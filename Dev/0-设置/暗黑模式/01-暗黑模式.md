# 功能实现：暗黑模式 (Dark Mode)

## 核心原理
现代 Web 应用的暗黑模式通常通过在 `<html>` 或 `<body>` 标签上添加一个 `dark` 类名来实现。
- **Tailwind CSS**: 当检测到父级有 `dark` 类时，会自动应用 `dark:` 前缀的样式 (例如 `dark:bg-slate-900`)。
- **shadcn/vue**: 组件已经内置了对 `dark` 类的支持 (通过 CSS 变量)。
- **VueUse**: 我们使用 `useDark` 这个组合式函数来自动管理这个 `dark` 类，并持久化存储用户的偏好。

## 步骤 2: 添加 UI 组件
我们需要一个漂亮的按钮来切换模式。shadcn/vue 提供了现成的 Button 组件。
在终端执行以下命令：
```bash
npx shadcn-vue@latest add button
```

## 步骤 3: 创建切换组件
我们将创建一个独立的组件来管理暗黑模式切换。

**操作**:
创建文件 `src/components/settings/ModeToggle.vue`，并填入以下代码：

```vue
<script setup lang="ts">
import { useDark, useToggle } from '@vueuse/core'
import { Button } from '@/components/ui/button'

const isDark = useDark()
const toggleDark = useToggle(isDark)
</script>

<template>
   <Button variant="outline" size="icon" @click="toggleDark()">
      <span :class="isDark ? 'icon-[lucide--moon]' : 'icon-[lucide--sun-dim]'" class="w-5 h-5"></span>
   </Button>
</template>
```

## 步骤 4: Home页面中使用

**操作**:
修改 `src/views/Home.vue`，引入并使用这个组件：

```vue
<script setup lang="ts">
import ModeToggle from "../components/settings/ModeToggle.vue";
</script>

<template>
    <div class="h-full flex flex-col p-8 space-y-8 animate-in fade-in slide-in-from-bottom-4 duration-500">

        <!-- 快速设置区域 -->
        <div class="grid gap-4 md:grid-cols-2 max-w-4xl mx-auto w-full">
            <!-- 1. 外观设置 -->
            <div
                class="flex items-center justify-between p-4 border rounded-xl bg-card/50 backdrop-blur-sm hover:bg-card/80 transition-colors shadow-sm">
                <div class="flex items-center gap-3">
                    <div class="flex p-2 bg-primary/10 rounded-lg text-primary">
                        <div class="icon-[lucide--sun] w-5 h-5"></div>
                    </div>
                    <div>
                        <h3 class="font-medium">外观主题</h3>
                        <p class="text-xs text-muted-foreground">深色 / 浅色模式</p>
                    </div>
                </div>
                <!-- 切换暗黑模式 -->
                <ModeToggle />
            </div>

        </div>
    </div>
</template>
```

## 步骤 5: 暗黑模式持久化
**src/App.vue**:
```typescript
import { useDark } from '@vueuse/core';

// 初始化暗黑模式 (这就够了，它会自动生效并持久化)
useDark();
```
