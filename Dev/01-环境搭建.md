# 环境搭建

## 1. 初始化 Tauri 项目
执行以下命令创建项目：
```bash
pnpm create tauri-app@latest .
```

**配置选项**:
✔ Package name · xgtools
✔ Identifier · com.tauri-app.xgtools
✔ Current directory is not empty, do you want to overwrite? · yes
✔ Choose which language to use for your frontend · TypeScript / JavaScript - (pnpm, yarn, npm, deno, bun)
✔ Choose your package manager · pnpm
✔ Choose your UI template · Vue - (https://vuejs.org/)
✔ Choose your UI flavor · TypeScript

```bash
pnpm install
pnpm tauri dev
```

## 2. 安装VueUse
安装 VueUse：
```bash
pnpm install @vueuse/core
```

## 3. 安装与配置 Tailwind

1.  **安装 Tailwind**:
```bash
pnpm add tailwindcss @tailwindcss/vite
```

2.  **配置 Vite 插件**:
```javascript
import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    tailwindcss(),
  ],
})
```

3.  **创建 CSS 文件**:
在 `src/style.css` 中添加 Tailwind 指令：
```css
@import "tailwindcss";
```

4. ** main.ts 引入 style.css**:
在 `src/main.ts` 中引入 `style.css`：
```typescript
import "./style.css";
```

## 4. 安装与配置 shadcn/vue (UI组件库)
shadcn/vue 是一个基于 Tailwind CSS 的组件库。

### 4.1 准备工作：配置路径别名 (@)
在初始化 shadcn 之前，必须确保项目支持 `@` 别名指向 `src` 目录。

1.  **修改 `tsconfig.json`**:
```json
{
  "compilerOptions": {
    //...

    /* 配置路径别名 */
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

2.  **修改 `vite.config.ts`**:
需要安装 `pnpm i -D @types/node` 以使用 `path` 模块。
```typescript
import path from "path"

export default defineConfig({
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
  // ...其他配置
})
```

### 4.2 初始化 shadcn/vue
执行 CLI 初始化命令：
```bash
pnpm dlx shadcn-vue@latest init
```
