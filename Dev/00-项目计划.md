# XGTools 桌面效率工具 开发计划

- **技术栈**: Tauri, Vue3, TypeScript, Vueuse, Tailwind, shadcn/vue.

## 工具目标
1. 键盘显示、键盘桌宠
2. 截图
3. 翻译
4. 格式转换

## 环境搭建
1. 初始化 Tauri 项目 (Vue3 + TS)。
2. 安装并配置 VueUse (Vue 组合式 API 工具库)。
3. 安装并配置 Tailwind CSS (v4 样式引擎)。
4. 安装并配置 shadcn/vue (UI 组件库)。
5. 初始化，清空默认样式、默认文件。
6. 安装图标 iconify
7. 配置 Git 仓库。
