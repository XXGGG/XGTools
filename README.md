## 简介

XGTools 是一个基于 Tauri 和 Vue 3 开发的桌面应用，集成了桌面宠物、截图、翻译等功能。

本项目仅供学习使用，不可商用。

## 功能

### 1. 基础设施
- 基于 Tauri v2 + Vue 3 + TypeScript 构建。
- UI 采用 Tailwind CSS v4 和 shadcn/vue 样式库。

### 2. 系统集成
- **开机自启**: 支持随系统启动自动运行。
- **暗黑模式**: 支持跟随系统主题及手动切换。
- **系统托盘**: 支持最小化到托盘及右键菜单。

### 3. 计划功能
- **桌面宠物**: 具有动画效果的交互式桌面伙伴。
- **截图与 OCR**: 屏幕截图并支持文字识别。
- **翻译**: 即时悬浮翻译功能。
